A full-stack Netflix-inspired streaming platform built with Next.js, Express, and MongoDB.

## Tech Stack
- Frontend: Next.js 14, Tailwind CSS, TypeScript
- Backend: Node.js, Express, JWT Auth
- Database: MongoDB Atlas
- API: TMDB

## Setup
1. Clone the repo
2. cd cinex-frontend && npm install && npm run dev
3. cd cinex-backend && npm install && npm run dev
4. Add your TMDB API key and MongoDB URI in .env files
